# HERMITAGE DECOR

Премиальный интернет-магазин мебели и предметов интерьера (HTML, CSS, JavaScript).

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Главная (hero, категории, слайдеры, о компании) |
| `catalog.html` | Каталог с фильтрами и сортировкой |
| `product.html` | Карточка товара |
| `cart.html` | Корзина и оформление заявки через WhatsApp |
| `account.html` | Регистрация, вход, профиль, история заявок |
| `favorites.html` | Избранное |
| `compare.html` | Сравнение характеристик (до 4 товаров) |

## WhatsApp

В `js/data.js` замените `whatsapp: '79001234567'` на номер менеджера (только цифры, код страны без +).

## Оплата

Онлайн-оплаты нет. Корзина формирует заявку и открывает WhatsApp с готовым текстом.

## Данные

Товары и настройки — в `js/data.js`. Корзина, избранное и пользователи хранятся в `localStorage` браузера.
